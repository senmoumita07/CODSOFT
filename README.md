# 🚢 Titanic Survival Prediction

This project is a machine learning-based analysis of the famous Titanic dataset. The goal is simple but meaningful — to predict whether a passenger survived the disaster based on available data like age, gender, class, and fare.

## 📌 Project Overview**

The Titanic dataset is one of the most well-known beginner datasets in data science. In this project, I explored the dataset, cleaned the data, created new features, and built a predictive model to understand survival patterns.

This project helped me get hands-on experience with real-world data and understand how machine learning models work step by step.

## 🔧 What I Did

- Cleaned and handled missing data (Age, Embarked, etc.)
- Converted categorical data into numerical format
- Created new features like:
  - **FamilySize** (number of family members onboard)
  - **IsAlone** (whether the passenger was alone or not)
- Visualized the data using graphs to understand patterns
- Built and trained machine learning models:
  - Logistic Regression
  - Random Forest (for comparison)

## 📈 Model Performance

The model achieved an accuracy of around **82%**, depending on the algorithm used.
